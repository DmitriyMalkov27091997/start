1. Установить дополнительный софт:
        * ActivePerl-5.16.1.1601-MSWin32-x64-296175
        * Visual Studio Code
        * Git for Windows
2. Открыть окно "Свойства системы", перейти во вкладку "Дополнительно" и нажать кнопку "Переменные среды..."
3. Выделить переменную "Path" и нажать кнопку "Изменить"
4. Добавить две новых переменных в которых необходимо указать путь к следующим папкам:
        * ..\AM800_Firmware\tools\make-3.81\bin
        * ..\AM800_Firmware\tools\openocd\bin
5. Сохранить изменения и перезагрузить компьютер.
6. Открыть Visual Studio и установить следующие расширения:
        * ms-vscode.cpptools
        * marus25.cortex-debug
        * ms-vscode.makefile-tools
7. Открыть файл "makefile" и скорректировать путь указанный в переменной "GCC_DIR" 
                GCC_DIR = (путь к папке)/AM800_Firmware

Для выполнения сборки и очистки проекта откройте терминал "CTRL+~"
                * Сборка make all
                * Очистка make clean

Для налача отладки нажмите F5
